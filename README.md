# mBot

A Discord bot that listens for `.np` or `.fm` commands and automatically adds the currently playing song (reported by [.fmbot](https://fmbot.xyz/)) to a Spotify playlist.

## Discord Bot Setup

- Create a new application in the [Discord Developer Portal](https://discord.com/developers/applications).
- In the **Bot** tab, enable the **Message Content Intent** under *Privileged Gateway Intents*.
- In the **Bot** tab, click **Reset Token** to generate a token. Save this for later.
- In the **Installation** tab under **Default Install Settings** > **Guild Install**, add the `bot` scope and the following permissions:
  - Add Reactions
  - Manage Messages
- Copy the *Install Link* from the **Installation** tab, open it in your browser, and add the bot to your server.

## Spotify App Setup

> As of recent Spotify policy changes, only **Spotify Premium** users can create developer apps, and each account is limited to **one app**.

- Go to the [Spotify Developer Dashboard](https://developer.spotify.com/dashboard) and create a new app.
- Add a redirect URI. Since Spotify no longer allows `http://localhost` for new apps, use an existing HTTPS URL you have access to (e.g. `https://yourdomain.com/spotify-callback`). The page doesn't need to load, you just need to copy the URL from your browser's address bar after the redirect.
- Note your **Client ID** and **Client Secret** from the app settings page.
- Run the auth script (see below) to generate a refresh token.

## Configuration

Fill in the values in `.env`:

```ini
DISCORD_TOKEN=         # Your Discord bot token
SPOTIFY_CLIENT_ID=     # From the Spotify Developer Dashboard
SPOTIFY_CLIENT_SECRET= # From the Spotify Developer Dashboard
SPOTIFY_REFRESH_TOKEN= # Generated by running node spotify-auth.js
SPOTIFY_PLAYLIST_ID=   # The ID from your Spotify playlist URL
SPOTIFY_REDIRECT_URI=  # The redirect URI you registered in the Spotify app
ADDED_REACT=           # Emoji to react with when a song is added (default: ➕)
DUPLICATE_REACT=       # Emoji to react with when a song is already in the playlist (default: ♻️)
```

The playlist ID is the string after `/playlist/` in a Spotify playlist URL:
```
https://open.spotify.com/playlist/59kzZROolqB5PUGrBkszTM
                                  ^^^^^^^^^^^^^^^^^^^^^^
```

For `ADDED_REACT` and `DUPLICATE_REACT`, you can use any standard emoji or a custom Discord emoji. To get the custom emojis value, right-click it in Discord and click **Copy Text**. 

It will copy something like `<:poggies:123456789012345678>`. Strip the angle brackets and starting colon and use `poggies:123456789012345678` in the env file.

## Auth Setup

Run the auth script once to link your Spotify account and save a refresh token to your `.env`:

```
node spotify-auth.js
```

It will print an authorization URL. Open it in your browser, log into Spotify, then copy the full URL from the address bar after the redirect and paste it back into the terminal. The page won't load but that's fine, all you need is the URL.

## Running

```
npm install
node app.js
```
